\# BunnyEra Public Assets



This folder stores public-facing assets that can be used in BunnyEra websites, documentation, product pages, and marketing materials.



\## Asset Types



Examples:



\- Website banners

\- Product preview images

\- Public illustrations

\- Marketing visuals

\- Social media images

\- Landing page graphics

\- Demo screenshots



\## Usage



Public assets can be used in:



\- BunnyEra official websites

\- BunnyEra Docs

\- BunnyEra Console

\- GitHub README files

\- Product launch pages

\- Social media posts



\## Rules



\- Only store public-safe assets here.

\- Do not include private customer data.

\- Do not include contracts or confidential screenshots.

\- Do not include API keys or credentials.

\- Use clear filenames.

\- Keep high-quality source files organized.
