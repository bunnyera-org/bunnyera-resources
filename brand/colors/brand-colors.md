\# BunnyEra Brand Colors



This file defines the core BunnyEra brand colors.



\## Primary Colors



| Name | Hex | Usage |

|---|---|---|

| Bunny Dark | `#020617` | Main dark background |

| Bunny Emerald | `#10B981` | Primary accent color |

| Bunny Cyan | `#22D3EE` | AI and technology accent |

| Bunny Pink | `#F9A8D4` | Momo IP and friendly brand accent |

| Bunny White | `#FFFFFF` | Text and highlights |



\## Usage Rules



\- Use dark backgrounds for AI, console, cloud, and security products.

\- Use emerald for primary buttons, success states, and active highlights.

\- Use cyan for AI, automation, network, and futuristic technology visuals.

\- Use pink carefully for Momo, friendly onboarding, cute UI moments, and brand warmth.

\- Keep interfaces clean, modern, and readable.

\- Do not use too many bright colors on the same screen.
