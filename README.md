# BunnyEra Resources

This repository contains public brand assets, UI elements, and visual resources for the BunnyEra ecosystem.  
It provides open, reusable materials for documentation, community use, and ecosystem development.

## Contents
- Logos and brand marks
- UI components and icons
- Wallpapers and visual assets
- Mascot illustrations (BunBun & Mr. Bear)
- Public documentation assets

## Usage
All assets in this repository are intended for public use within the BunnyEra ecosystem.  
Commercial use outside the ecosystem may require permission.
